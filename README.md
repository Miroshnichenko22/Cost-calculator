---

### Стек / Stack: 

* HTML
* CSS
* Javascript 

---

### Этапы разработки: 

* Калькулятор рассчитывает общую стоимость поездки и выводит информацию в поле заказа / 
The calculator calculates the total cost of the trip and displays the information in the order field
